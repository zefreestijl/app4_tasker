J.2. Quality Checks

Checks Definition Responsible Party Software Program(s) Frequency
Visual Check Ensure there are no unintended model components and the design intent has been followed.
Interference Check Detect problems in the model where two building components are clashing including soft and hard.
Standards Check Ensure that the BIM and AEC CADD Standard have been followed (fonts, dimensions, line styles, levels/layers, etc).
Model Integrity Checks Describe the QC validation process used to ensure that the Project Facility Data set has no undefined, incorrectly defined or duplicated elements and the reporting process on non-compliant elements and corrective action plans.
