3D Coordination / Clash Detection

A process in which model elements are analyzed using Clash Detection software to highlight potential installation conflicts. The goal of clash detection is to update the design to eliminate potential system collisions prior to starting on site.

Potential Value

  • Coordinate building project through a model.
  • Reduce and eliminate field conflicts, significantly reducing RFIs compared to other methods.
  • Visualize construction.
  • Increase productivity on site.
  • Reduced construction cost; potentially less cost growth (i.e., fewer change orders).
  • Decrease construction time.
  • More accurate as-built drawings.
Resources Required Team Competencies Required
  • Design Authoring Software
  • Model Review application (e.g., Navisworks, Solibri)
  • Proficiency in Design Authoring Software
  • Ability to manipulate, navigate, and review a 3D model
  • Strong understanding of constructibility and how building/facility systems integrate with one another
