Responsibility Assignment (RACI)

This RACI (pronounced Ray-See) matrix defines the participation by various roles in completing tasks or deliverables for the project. It clarifies boundaries and ensures that every action item has clear ownership.

Legend:

HIGHLIGHT TYPE DESCRIPTION ALLOCATION
R Responsible Undertaking the actual activity or work. 1 Team Member per task
A Accountable Ultimate owner of completion; "the buck stops here." Exactly 1 Team per task
C Consulted Two-way communication; provides input or expertise. No maximum or minimum
I Informed One-way communication; kept updated on progress. No maximum or minimum

BIM Management Rule: While many can be Responsible, only one entity can be Accountable for a single task to avoid confusion and project delays.
