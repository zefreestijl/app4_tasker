3D Coordination / Clash Detection

A process in which model elements are analyzed using Clash Detection software to highlight potential installation conflicts.

The goal of clash detection is to update the design to eliminate potential system collisions prior to starting on site.

Potential Value

  • Coordinate building project through a model
  • Reduce and eliminate field conflicts; which reduces RFI's significantly compared to other methods
  • Visualize construction
  • Increase productivity
  • Reduced construction cost; potentially less cost growth (i.e. less change orders)
  • Decrease construction time
  • Increase productivity on site
  • More accurate as built drawings

Resources Required

  • Design Authoring Software
  • Model Review application

Team Competencies Required

  • Design Authoring Software knowledge
  • Ability to manipulate, navigate, and review a 3D model
  • Strong understanding of constructibility and how building/facility systems integrate with one another
