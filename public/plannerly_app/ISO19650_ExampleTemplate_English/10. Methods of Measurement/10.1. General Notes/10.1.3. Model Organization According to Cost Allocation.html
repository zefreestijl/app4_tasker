Model Organization According to Cost Allocation

In line with identification parameters, when categorizing the project into different cost allocation sectors, models should be structured to distinguish and quantify elements based on their respective locations, such as Building Number, Core & Shell, FitOut, Below and Above Ground, etc.

This segregation can be accomplished through several methods, including:

  • Distinct Files: Model each building, project sector, or location in distinct files.
  • Containers/Worksets: If using a single model file, assign different containers (e.g., Worksets in Revit) to each sector or location.
  • Instance Parameters: Create/provide an instance parameter to all applicable elements that require specific cost allocation based on the building / sector / location.
