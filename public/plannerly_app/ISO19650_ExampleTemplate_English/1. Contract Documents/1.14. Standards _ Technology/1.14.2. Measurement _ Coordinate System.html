Measurement and Coordinate System

This section establishes the common geometric framework for the project to ensure accurate alignment and interoperability.

Coordinate Rules

  • All models of the building will use the Location at Latitude: 0°, Longitude: 0° in the Architectural model as a shared base point that must be used by the entire project team for building coordination purposes.
  • A .dwg file named "GRIDS" locates the building structural grid lines relative to the project origin.
  • All models of site utilities will use a coordinate system separate from the building coordinate system and will be based on the Civil Site drawings.
  • The General Contractor will coordinate the placement of this point first and all other models will follow.
  • Each model will be aligned and rotated so that when exporting to shared formats, they align without requiring manual moving or rotation.
  • The coordination process will be documented and published for the entire team to access. This allows for the sharing and use of coordinate point data between all trades for field location and installation.
Note to all Revit users

The origin marker, “Base Point”, and “Shared Coordinate” should all be in the same place in your models. You can then use the site/civil information to “locate” the project for solar studies, heat days, lighting, etc.
