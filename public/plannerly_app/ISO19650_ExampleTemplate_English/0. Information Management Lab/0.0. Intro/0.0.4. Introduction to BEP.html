Introduction: BIM Execution Plan (BEP)

What is a BEP?

The BIM Execution Plan (BEP) explains how the project team will deliver information in response to the EIR. It sets out the processes, standards, responsibilities, and delivery approach that will be followed to produce, coordinate, review, and verify information throughout the project lifecycle.

Who is this for?

  • Lead Appointed Parties: (e.g., General Contractor, Lead Architect) responsible for coordinating information delivery.
  • Task Teams: (e.g., MEP Subs, Structural Engineers) producing models, data, and documentation.
  • Information Managers: Overseeing compliance with the EIR.
Document Usage Guide

This BEP is a response document. It must be authored by the Lead Appointed Party and approved by the Appointing Party (Client). Unlike the EIR which states "what" is needed, this document defines "how" it will be achieved using specific tools, workflows, and standards.
